Submission procedure

  1. Confirm both predictions files exist (see the orange box above). Make sure both HF models are still public.
  2. Open the Leaderboard Space and click the 😀 Submit tab. (Log into HF if the form asks you to.)
  3. Submit the Strict entry: select Track = Strict, fill the per-track + shared fields above, upload prabhasa-b_s_strict_preds.json in the Results file (JSON) field, then click Submit Results.
  4. Submit the Strict-Small entry: repeat with Track = Strict-small, the Strict-Small per-track values, and prabhasa-b_ss-0.1_strict-small_preds.json.
  5. Wait for the “Submission successful” message for each; note the confirmation / timestamp. Both entries should appear on the respective track tabs.
If the form warns about missing/incomplete tasks (e.g. EWoK or a GLUE task), that is fine — incomplete submissions are allowed and missing tasks score 0. You can re-run the gap and re-submit later; resubmission is encouraged.
